LinkedIn - Bot_1

Bot_1 is an automated connect-request-bot for LinkedIn. The connect-request-bot works by replicating the process of searching a category of users on LinkedIn and sending a connection-request to the first 8 users who show up in that search. This user process runs on a loop wich can be configured by the user to be run as many times as desired. This program acts as an automated self-marketing campaign and can be reconfigured for any web app with an available desktop-platform

    Tech used:

  • Selenium IDE

LinkedIn - Bot_2

Bot_2 is an automated easy-apply-bot for LinkedIn. The easy-apply-bot works by replicating the user's process of apllying to jobs on LinkedIn and running that replication on a loop. The amount of times you decide to run the loop then determines the amount of applications you want to send out. This program acts as an automated self-marketing campaign and cannot currently be reconfigured for other web based platforms.

    Tech used:

  • Selenium IDE

Trading-Bot

The trading-bot is a crypto market trading application that repeatedly scans a given crypto market value and depending on the value change with each scan, will either trigger a hold, sell or purchase of the given crypto market
